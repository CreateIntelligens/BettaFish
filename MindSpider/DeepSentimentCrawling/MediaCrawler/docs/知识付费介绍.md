# 知識付費介紹
開源是一種無私奉獻，從MediaCrawler開源到現在有一年多，它並沒有帶給我多少實質性的東西，就拿收入來說，贊助費、讚賞等等全部加起來還沒有之前一個月的薪水。

後面搞了MediaCrawler源碼剖析課程之後，收入稍微好一點，但也是羣裏兄弟對我開源的支持，在此也非常感謝你們～

但是我依然堅持持續開源，從開始的xhs、dy 2個平臺支持，到現在已經有**7個平臺**支持，每一次增加一個平臺其實都會耗費很大的時間去寫代碼和調試代碼。。。。

在今天跟一個羣裏好朋友聊天，他說：開源開發者也要活下去。你不要不好意思做知識付費，你的勞動是有價值的。

他點醒我了，因此我把我所提供的知識付費內容放在下面，有需要的朋友可以看看～
